-Install packages using pip3:
pip list => list all packages globally installed
pip3 install prettytable
pip3 install colorgram.py
pip install psycopg2 => Install support for postgres

-Create virtual environment:
virtualenv -p python <name> Ej: virtualenv -p python env

-Activate environment:
Set path to activate file and hit enter: venv/bin/activate

-Deactivate virtual environment:
deactivate


-Install packages saved in txt file:
 pip install -r requirements.txt
